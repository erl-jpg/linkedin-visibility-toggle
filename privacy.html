Privacy Policy — LinkedIn Visibility Toggle

Last updated: July 7, 2026

LinkedIn Visibility Toggle is a Chrome extension that opens LinkedIn's "Profile viewing options" settings page and selects one of LinkedIn's three profile visibility modes for you.

What data the extension handles

The extension does not collect, transmit, or sell any personal data.

The only data it stores is a temporary record of which visibility mode you just clicked (for example, "private mode"). This is saved locally in your browser using Chrome's storage.local API so the extension can select the correct option after LinkedIn's settings page finishes loading. It is automatically removed once the option has been applied.

What the extension can access

The extension only runs on this single LinkedIn page:

https://www.linkedin.com/mypreferences/d/profile-viewing-options

It cannot read or change your messages, connections, profile content, cookies, passwords, browsing history, or any other website or LinkedIn page.

Data sharing

No data is ever sent to the developer or any third party. There are no analytics, trackers, or external servers.

Contact

Questions or privacy requests: please open an issue at github.com/erl-jpg/linkedin-visibility-toggle/issues.
